Sales Table

This table records information about the sales transactions of the drugs.

Column Name Data Type Description
Sales_ID int Primary Key, Auto Increment
Drug_ID int Foreign Key referencing Drugs(Drug_ID)
Sale_Date date Date of the Sale
Quantity int Number of Units Sold
Discount decimal(5,2) Discount on the Sale
Price int Price per Unit
Cut_ID int Foreign Key referencing Emp_Cut(Cut_ID)
Loc_ID int Foreign Key referencing Location(Loc_ID)
Total_Price int Total Price of the Sale
